# Initiation a jsp
## Description

Ce projet est une application JSP démontrant la gestion des formulaires. Il permet à l'utilisateur de remplir un formulaire et de soumettre les données vers une page de traitement qui affiche les informations saisies.

## Fonctionnalités

- Formulaire HTML avec plusieurs champs de saisie
- Traitement des données POST en JSP
- Affichage des données soumises sur une page de résultat

## Structure du projet

```
├── index.jsp      # Page contenant le formulaire
└── value.jsp        # Page affichant les données saisies
```

## Utilisation

1. Accédez à la page `index.jsp`
2. Remplissez le formulaire avec vos données
3. Cliquez sur "Soumettre"
4. Vous serez redirigé vers `value.jsp` qui affichera les données

## Prérequis

- Apache Tomcat 9.0.115
- JDK 8 ou supérieur
